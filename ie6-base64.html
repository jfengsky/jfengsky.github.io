公司项目踩到个大坑，要在ie6显示base64编码的图片...

方法一

在CSDN里一个帖子找到一个办法，利用MHTML， 代码如下：

<!--
------=_NextPart_000_0000_01CA9F59.AFB45FE0
Content-Type: image/jpg
Content-Transfer-Encoding: base64
Content-Location: logo.jpg
//base64 code

!-->
<img src="data:image/jpg;base64, base64 code..." onerror="src='mhtml:' + document.location.href + '!logo.jpg';"/>
方法二
<!--
Content-Type: multipart/related; boundary="[MHTML_EMBED_CODE]"

--[MHTML_EMBED_CODE]
Content-Type: image/jpg
Content-Location: img1
Content-Transfer-Encoding: base64

// base64 code 1


--[MHTML_EMBED_CODE]
Content-Type: image/png
Content-Location: img2
Content-Transfer-Encoding: base64

// base64 code 2

--[MHTML_EMBED_CODE]
Content-Type: image/png
Content-Location: img3
Content-Transfer-Encoding: base64

// base64 code 3

-->

<html>
<head>
<meta http-equiv="content-type" content="text/html; charset=GB2312">
</head>
<body>


<img id="img1" src="*" />
<img id="img2" src="*" />
<img id="img3" src="*" />

<script type="text/javascript">
var img1=document.getElementById("img1"),
    img2=document.getElementById("img2"),
    img3=document.getElementById("img3");

img1.src='mhtml:'+document.location+'!img1';
img2.src='mhtml:'+document.location+'!img2';  
img3.src='mhtml:'+document.location+'!img3';
</script>
</body>
</html>
